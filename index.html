Git & GitHub Practice Repository

Basic Commands

  • git init: Initialize a new Git repository.
  • git status: Check the current state of the repository.
  • git add [file]: Add a file to the staging area.
  • git commit -m "message": Commit changes with a message.
  • git log: View commit history.

Branching

  • git branch [branch-name]: Create a new branch.
  • git checkout [branch-name]: Switch to a different branch.
  • git merge [branch-name]: Merge a branch into the current branch.

Remote Commands

  • git remote add origin [URL]: Link your local repo to a remote repo.
  • git push origin [branch-name]: Push your branch to the remote repository.
  • git pull origin [branch-name]: Fetch and merge changes from the remote repository.

Other Useful Commands

  • git clone [URL]: Clone an existing repository.
  • git restore [file]: Discard changes in the working directory.
  • git reset --hard HEAD: Reset your working directory to the last commit.
